🧮 Projet 01 – Calculatrice en ligne de commande
🎯 Objectif
Créer une calculatrice simple en ligne de commande qui permet à l'utilisateur de choisir une opération (addition, soustraction, multiplication, division), d'entrer deux nombres, et d'afficher le résultat.

🛠️ Compétences visées
* Entrées et sorties (input, print)
* Conditions (if, elif, else)
* Fonctions
* Gestion des erreurs (try, except)

📋 Fonctionnalités attendues
* Menu interactif pour choisir l'opération
* Saisie de deux nombres par l'utilisateur
* Calcul de l'opération choisie
* Affichage du résultat
* Possibilité de recommencer ou quitter

🔁 Étapes de réalisation
* Afficher un menu d'opérations
* Lire le choix de l'utilisateur
* Demander deux nombres
* Effectuer l'opération correspondante
* Afficher le résultat
* Gérer la division par zéro
* Gérer les erreurs de saisie (lettres, etc.)
* Ajouter une boucle pour répéter les calculs
